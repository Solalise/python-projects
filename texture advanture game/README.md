# 🎮 Texture Adventure Game

## 📘 프로젝트 소개
텍스트 기반으로 진행되는 간단한 어드벤처 게임입니다.
플레이어는 마을에서 시작해 몬스터를 사냥하고, 장비를 장착하고,
레벨업하며 성장해 나가는 시스템을 가지고 있습니다.
전투, 아이템, 스킬, 인벤토리 등 RPG 요소가 간단하게 구현되어 있습니다.

## 🛠️ 사용 기술
- Python 3
- 조건문, 반복문, 함수, 딕셔너리, 리스트 활용
- 콘솔 입력 기반 인터페이스

## 🧩 주요 기능
- 플레이어 스탯(HP, MP, 공격력, 방어력) 관리
- 몬스터 전투 및 경험치 획득
- 장비 장착 / 해제 시스템
- 인벤토리 및 상점 기능
- 간단한 맵 이동 시스템
